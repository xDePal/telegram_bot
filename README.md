# telegram_bot

Функционал телеграм бота:
Пользователь пишет сообщение с вопросом касаемо продукции центра красок
ИИ модель на основании поиска по векторной базе данных дает ответ пользователю

План разработки: 
Шаг 0: 
Собрать и структурировать данные с сайта

Шаг 1: 
Создать телеграм бота

Шаг 2: 
Создать векторную базу данных
Создать ИИ-агента для поиска информации с помощью векторной базы данных

Применить: 
1) chunking + overlap при разделении данных
2) tempreture снизить до 0 для уменьшения галлюцинаций
3) в качестве моделей использовать модели от провайдера OpenRouter или Groq
4) так же включить guardrails в system prompt

Шаг 3: 
Сделать тесты с помощью LangSmith для оценки точности ответов модели
Использовать LLM as a judge для фильтрации ответов

Заметка: 
Просто перенести базу данных хрома дб без векторизации снова (при деплое)
