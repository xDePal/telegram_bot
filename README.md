# telegram_bot
AI-ассистент для Центр красок №1

Интеллектуальный Telegram-бот для автоматизации клиентской поддержки сети магазинов лакокрасочных материалов.

Бот способен отвечать на вопросы клиентов о компании, актуальных вакансиях, товарах и услугах — используя данные, загруженные с сайта.

## Стек технологий:
• Python — основной язык разработки
• LangChain — оркестрация LLM-пайплайна
• Groq API + Qwen3-32B — сверхбыстрый LLM-инференс
• ChromaDB — векторная база данных для семантического поиска
• HuggingFace (intfloat/multilingual-e5-base) — многоязычные эмбеддинги (локальная модель)
• python-telegram-bot — интеграция с Telegram

## Архитектура:
Реализована двухэтапная RAG-система (Retrieval-Augmented Generation): входящий запрос пользователя сначала переформулируется LLM для повышения точности поиска, затем выполняется семантический поиск по векторной базе, после чего финальный ответ генерируется с учётом найденного контекста. Это позволяет боту корректно обрабатывать как чёткие, так и размытые пользовательские запросы.
